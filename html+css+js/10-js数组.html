数组函数-学习目标

  • push/pop
  • unshift/shift
  • indexOf
  • slice
  • splice

push/pop

  • push就是在数组结尾添加数据
  • pop就是删除数组结尾的数据

indexOf

  • 找到该数据在数组的下标

slice

  • 从已有的数组中返回选定的元素。
  • 该方法不会改变原始数组。
  • 第一个参数是要拷贝这个数组的起始位置
  • 第二个参数是要拷贝这个数组的结束位置
  • 结果返回这个被截取的数组，不传参数就是整一个数组拷贝

splice

  • 从数组中添加/删除项目，然后返回被删除的项目。
  • 该方法会改变原始数组。
  • 第一个参数是添加/删除项目的位置
  • 第二个参数是要删除的项目数量
  • 第三个参数是向数组添加的新项目，可以不填
