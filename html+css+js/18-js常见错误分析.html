常见错误

依赖错误

Uncaught ReferenceError: "某个变量" is not defined

什么时候会出现这个错误？

依赖错误

如果在使用这个变量x的时候，在作用域里面找不到这个x，那么js引擎就会抛出ReferenceError

什么时候会出现这个错误？

语法错误

类型错误

Uncaught TypeError: Cannot read property '某个属性' of undefined

什么时候会出现这个错误？

类型错误

当你在对这个变量进行不合理的操作时，比如非函数却用()进行函数调用，或者引用undefined类型的值得属性，那么js引擎就会抛出TypeError
