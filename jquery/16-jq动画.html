jq动画

数字参数是动画时间，还可以加回调函数

  • 隐藏:$('#box').hide(400, function(){ });
  • 显示:$('#box').show(400, function(){ });
  • 隐藏显示切换:$('#box').toggle(400, function(){ });
  • 渐渐出现:$('#box').fadeIn(400, function(){ });
  • 渐渐隐藏:$('#box').fadeOut(400, function(){ });
  • 渐渐出现隐藏切换:$('#box').fadeToggle(400, function(){ });
  • 渐渐出现到指定的透明度:$('#box').fadeTo(400,0.5, function(){ });
  • 自定义动画:$('#box').animate({'width':600,'height':600}, 1000, function () { });
